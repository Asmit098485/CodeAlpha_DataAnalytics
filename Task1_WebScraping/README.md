# 📚 CodeAlpha Data Analytics Internship

## 📝 Task 1: Web Scraping using Python

### 📌 Objective

The objective of this project is to extract book information from a public website using Python Web Scraping techniques. The scraped data is stored in a CSV file and analyzed using Pandas and Matplotlib.

---

## 🌐 Website Used

https://books.toscrape.com/

---

## 🛠️ Technologies Used

- Python 3.13
- Requests
- BeautifulSoup4
- Pandas
- Matplotlib
- Visual Studio Code

---

## 📂 Project Structure

```

│
├── web_scraping.py

├── books.csv

├── README.md
└── screenshots/
```

---

## 📊 Data Collected

The following information was extracted:

- Book Title
- Price
- Availability

---

## ⚙️ Steps Performed

1. Sent an HTTP request to the website.
2. Parsed the HTML using BeautifulSoup.
3. Extracted the required book information.
4. Saved the data into a CSV file.
5. Loaded the CSV file using Pandas.
6. Cleaned the Price column.
7. Performed basic data analysis.
8. Created a histogram showing the distribution of book prices.

---

## 📈 Output

- Successfully scraped data from the website.
- Created `books.csv`.
- Performed data analysis.
- Visualized the price distribution using Matplotlib.
