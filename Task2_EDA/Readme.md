# 📊 Task 2: Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Objective

# 📋 Exploratory Data Analysis Performed

The following analyses were performed:

- Loaded the dataset
- Displayed first and last five records
- Checked dataset shape
- Explored column names
- Identified data types
- Displayed dataset information
- Detected missing values
- Checked duplicate records
- Generated descriptive statistics
- Analyzed passenger survival
- Analyzed passenger class distribution
- Analyzed gender distribution
- Analyzed embarkation ports
- Calculated average age
- Calculated fare statistics
- Generated correlation matrix
- Created multiple visualizations

---

# 🔍 Key Findings

- The Titanic dataset contains passenger information including age, gender, ticket fare, passenger class, and survival status.
- Missing values are present in the Age, Cabin, and Embarked columns.
- Most passengers did not survive.
- Third-class passengers were the highest in number.
- Male passengers outnumbered female passengers.
- Ticket fares vary significantly among passengers.
- The dataset is suitable for predictive analysis after preprocessing.
